About Open Graphing Calculator

Open Graphing Calculator began as an exercise that was used to learn a little about creating web applications. The goal was to make use of existing libraries to create something useful in a short amount of time. Feel free to fork the source on GitHub, submit issues, and contribute.

Source On GitHub Submit an Issue

We know there are many issues still to address, please report anything you might find on GitHub. Furthermore, if you find this resource useful, go ahead and share the link. Thank you.
